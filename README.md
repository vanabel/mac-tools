### English <span id="english"></span>

<details open>
<summary><b>🇬🇧 English</b></summary>

#### Tools

| Tool | Description |
|------|-------------|
| [**pdfmerge**](./pdfmerge/) | Merge PDFs with bookmarks (书签) and natural sort order |

#### Setup

Add to your `~/.zshrc`:

```bash
# pdfmerge
source ~/development/mac-tools/pdfmerge/pdfmerge
```
